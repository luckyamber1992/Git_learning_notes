## GitHub account
- 删除远程git仓库
git remote rm origin
- 删除本地仓库
 ls -a  #找到目录下.git
 rm -rf  .git   #删除
- git token重新配置
[token 配置](https://blog.csdn.net/qq_36160277/article/details/119785943)
- 源引用规格 master 没有匹配
  - 查看远程分支
  git branch -a
  - 本地分支的名称，要与远程分支的名称保持一致
